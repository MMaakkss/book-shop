# Vue 3 + TypeScript

In this project, I implemented an online bookstore. The data is stored in a fire store.

## Used technologies:

- Vue 3
- Pinia
- TypeScript
- Firebase
- Vue router
- Vue meta
- Vite

## How to run the project:

1. Clone this repo
2. Run `npm i`
3. Run `npm run dev`

#### Note

I don't provide .env file with database settings
